# Stepwise Regression & Factor Analysis of World Happiness Data Using SPSS
Countries nowadays focus more on boosting citizens’ happiness. However, the term happiness is broad and complex as it is subject to people’s definition of happiness which is indicated by various factors. Therefore, it would be interesting to understand the complexity of factors like GDP, social support, health, and corruption in predicting happiness using stepwise regression and factor analysis. The analysis was conducted using SPSS.
